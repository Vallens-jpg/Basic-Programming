Repositori ini berisi kumpulan kode program dasar menggunakan bahasa pemrograman Java. Proyek ini mendokumentasikan proses pembelajaran materi perkuliahan Basic Programming yang diampu oleh Bpk Deny, mulai dari pengenalan sintaks dasar hingga implementasi logika pemrograman yang lebih kompleks.

## 🚀 Fitur Utama

Proyek ini mencakup berbagai materi dasar pemrograman yang terbagi menjadi beberapa bagian:

* **Pengenalan Dasar:** Output teks sederhana dan perkenalan identitas.
* **Operasi Aritmatika:** Implementasi rumus matematika dasar seperti perhitungan luas persegi dan persegi panjang.
* **Logika Percabangan:** Penggunaan struktur kontrol `if`, `else if`, `else`, dan `switch case` untuk pengambilan keputusan.
* **Struktur Perulangan:** Implementasi berbagai jenis loop seperti `for`, `while`, dan `do-while`.
* **Logika Nested Loop:** Pembuatan pola visual seperti berbagai bentuk segitiga bintang.
* **Struktur Data:** Penggunaan Array satu dimensi dan Array multi-dimensi (matriks).

## 🛠️ Teknologi yang Digunakan

* **Java:** Bahasa pemrograman utama (menggunakan JDK 25).
* **IntelliJ IDEA:** Lingkungan pengembangan terpadu (IDE) yang digunakan untuk menyusun proyek ini.

## 📁 Susunan Project

Struktur folder dalam repositori ini adalah sebagai berikut:

```text
Basic-Programming/
├── .idea/                 # Konfigurasi project IntelliJ IDEA
├── src/                   # Folder source code utama
│   ├── array.java         # Implementasi array 1D dan 2D
│   ├── pertemuan_1.java   # Dasar output dan perkenalan
│   ├── pertemuan_2.java   # Variabel dan aritmatika dasar
│   ├── pertemuan_3.java   # Logika percabangan (IF/ELSE)
│   ├── pertemuan_4.java   # Struktur kontrol Switch Case
│   ├── pertemuan_5.java   # Dasar-dasar perulangan (Looping)
│   └── pertemuan_6.java   # Nested loop dan pola bintang
├── Basic Programming.iml  # File modul IntelliJ IDEA
└── .gitignore             # Daftar file yang diabaikan oleh Git
```
